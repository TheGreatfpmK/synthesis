CODE:

get-base-benchmark - runs base PAYNT on every model from models folder and stores logs to base-benchmark

storm-experiments - runs PAYNT with storm results to see potential improvements and stors logs to paynt-logs

Base settings: AR

------------------------------------------------------------

FOLDERS:

base-benchmark - folder that contains logs from runs of normal PAYNT that are used to compare potential improvement

models - contains all models used in the experiments and their respective properties

paynt-logs - log files from runs of PAYNT using the data from storm

results - storm results for every model used for the experiments

Some folders contain additional files that were used for more experiments
